## Decisions
- 2026-03-15: Phase 1 scoped to unblock 200+ seat enterprise procurement
- 2026-03-15: SAML 2.0 SP only (no IdP); rely on customer IdPs (Okta, Azure AD, OneLogin)
- 2026-03-15: SCIM 2.0 — Users + Groups resources; no custom schema extensions in v1
- 2026-03-15: API keys scoped to three levels: read, write, admin
- 2026-03-15: Audit forwarding sinks: webhook (with retry + DLQ) and S3/GCS object storage
- 2026-03-15: Rate limiting: sliding window, per-IP + per-user, 5 attempts per 15 minutes on login/2FA/recovery
- 2026-03-15: Encryption at rest: AES-256-GCM; key managed via GCP Secret Manager; TOTP secrets encrypted before DB write
